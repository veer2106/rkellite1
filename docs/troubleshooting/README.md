## 🆘 Quick Fixes

### Application Not Starting
1. Check if ports 3000 and 5001 are free
2. Verify database is running
3. Check environment variables
4. Review error logs

### Cannot Add Menu Items
1. Ensure you're logged in as Admin/Manager
2. Check all required fields are filled
3. Verify price is a valid number
4. Check backend logs for errors

### POS Not Showing Items
1. Check item availability is enabled
2. Verify section assignment
3. Check section filter selection
4. Refresh the page
